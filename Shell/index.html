Shell ​

简介 ​

Shell 是一个命令行解释器，它接收用户输入的命令并将其传递给操作系统执行。Shell 是 Linux 系统的重要组成部分，它提供了用户与操作系统交互的接口。

常用命令 ​

  • ls：列出当前目录下的文件和目录
  • cd：切换当前工作目录
  • pwd：显示当前工作目录的路径
  • mkdir：创建目录
  • rm：删除文件或目录
  • cp：复制文件或目录
  • mv：移动文件或目录
  • ln：创建链接
  • chmod：修改文件或目录的权限
  • chown：修改文件或目录的所有者
  • chgrp：修改文件或目录的所属组
  • tar：打包和解包文件
