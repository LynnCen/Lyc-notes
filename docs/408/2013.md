## 2. 带权路径长度

各结点权值x路径长度 求和 = 带权路径长度

带权路径长度最小即为哈夫曼树，每次从最小的权值开始构造

三叉树{2,3,4,5,6,7}构造哈夫曼树

（1）构造一个空结点0，分别与2，3相加的新结点权值 = 5 ，集合{4,5,5,6,7}

（2）4,5,5构造新结点 = 14，集合{6，7，14}

（3）6+7+14 = 27

故带权路径长度 = `(2+3)*3 + (4+5)*2 + (6+7)*1` = 46
