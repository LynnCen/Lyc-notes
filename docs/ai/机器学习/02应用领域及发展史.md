# 机器学习的应用领域与发展史

> 本章介绍机器学习的主要应用领域、发展历程及三大核心要素

---

## 一、机器学习的应用领域

### 1.1 核心应用场景

| 领域 | 说明 | 典型案例 |
|------|------|----------|
| 🖼️ 图像识别 | 计算机视觉的基础应用 | 人脸识别、物体检测、OCR |
| 🚗 无人驾驶 | 自动驾驶系统 | 萝卜快跑、特斯拉 FSD |
| 🌐 智能翻译 | 多语言实时翻译 | 科大讯飞翻译机、Google 翻译 |
| 📊 数据挖掘 | 从海量数据中发现规律 | 用户画像、推荐系统 |
| 🏥 医疗诊断 | 医疗影像分析与辅助诊断 | CT 影像分析、病理诊断 |

### 1.2 语音技术

::: info 语音技术双子星
| 缩写 | 全称 | 功能 | 方向 |
|------|------|------|------|
| **ASR** | Automatic Speech Recognition | 语音识别 | 语音 → 文字 |
| **TTS** | Text To Speech | 语音合成 | 文字 → 语音 |
:::

### 1.3 新兴应用方向

- 🎨 **3D 建模** - AI 辅助三维建模
- 🎤 **语音克隆** - 声音复刻技术
- 👤 **数字人** - 虚拟形象生成
- 🎬 **AIGC** - AI 生成内容（文本/图像/视频）

## 二、机器学习发展史

### 2.1 发展阶段时间线

```
1950s ──────── 1980s ──────── 2012 ──────── 2017 ──────── 2022 ──────── Now
  │              │              │              │              │           │
  ▼              ▼              ▼              ▼              ▼           ▼
符号主义      统计主义      神经网络      Transformer    ChatGPT     AIGC
(专家系统)    (SVM)        (AlexNet)      架构革命       横空出世    时代
```

### 2.2 四个发展阶段

#### 阶段一：符号主义（1950s - 1970s）

::: details 📜 关键事件
- **1950年**：图灵设计象棋程序
- **1956年**：达特茅斯会议，"人工智能"概念正式诞生 🎂
- **1962年**：IBM 跳棋程序战胜人类
:::

**局限性：** 难以处理图像、语音等复杂问题

#### 阶段二：统计主义（1980s - 2000）

| 年份 | 里程碑事件 |
|------|------------|
| 1993 | 支持向量机（SVM）提出 |
| 1997 | IBM "深蓝" 战胜国际象棋冠军卡斯帕罗夫 ♟️ |

#### 阶段三：神经网络（21世纪初 - 2016）

| 年份 | 里程碑事件 |
|------|------------|
| 2012 | **AlexNet** 在 ImageNet 竞赛中获胜，开启 CNN 时代 🏆 |
| 2016 | **AlphaGo** 基于强化学习战胜李世石 🎯 |

#### 阶段四：大规模预训练模型（2017 - 至今）

| 年份 | 里程碑事件 |
|------|------------|
| 2017 | Google 提出 **Transformer** 架构（划时代！） 🚀 |
| 2018 | GPT-1.0 和 BERT 发布 |
| 2022 | **ChatGPT** 横空出世 💥 |
| 2023 | 进入"百模大战"时代 |

::: warning 面试重点
**Transformer** 是 2017 年提出的革命性架构，奠定了大模型的基础，面试 100% 会考察！
:::

## 三、人工智能发展三要素

### 3.1 核心三要素

```
        ┌─────────────────────────────────┐
        │      🎯 人工智能发展三要素       │
        └─────────────────────────────────┘
                       │
       ┌───────────────┼───────────────┐
       ▼               ▼               ▼
   ┌───────┐      ┌───────┐      ┌───────┐
   │ 📊    │      │ 🧮    │      │ ⚡    │
   │ 数据  │      │ 算法  │      │ 算力  │
   └───────┘      └───────┘      └───────┘
   模型训练        解决问题       计算资源
   基础原料        数学方法       支撑保障
```

::: tip 三要素关系
- **数据**是基础 - 质量决定模型上限
- **算法**是核心 - 解决问题的数学方法
- **算力**是保障 - 支撑训练和推理
:::

### 3.2 算力类型对比

| 类型 | 全称 | 特点 | 适用场景 |
|------|------|------|----------|
| **CPU** | Central Processing Unit | 通用处理器 | 输入输出密集型任务 |
| **GPU** | Graphics Processing Unit | 并行计算能力强 | 深度学习训练 ⭐ |
| **TPU** | Tensor Processing Unit | Google 开发，张量优化 | 神经网络专用 |
| **NPU** | Neural Processing Unit | 神经网络处理器 | 华为昇腾芯片 |

::: warning 能源挑战
大模型训练耗能巨大，能源需求已成为发展的重要瓶颈。
:::
