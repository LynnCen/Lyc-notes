# 机器学习常用术语

> 本章介绍机器学习中的核心术语：样本、特征、标签，以及数据集划分方法

---

## 一、核心概念三要素

### 1.1 概念关系图

```
┌─────────────────────────────────────────────────────────┐
│                      数据集 (Dataset)                    │
├─────────────────────────────────────────────────────────┤
│  编号  │  学历  │  工作经验  │  工作地点  │  就业薪资   │  ← 特征列 (Features)
├────────┼────────┼────────────┼────────────┼─────────────┤     + 标签列 (Label)
│   1    │  本科  │   2年      │   北京     │   15000    │  ← 样本 (Sample)
│   2    │  硕士  │   3年      │   上海     │   20000    │  ← 样本 (Sample)
│   3    │  本科  │   1年      │   深圳     │   12000    │  ← 样本 (Sample)
│  ...   │  ...   │   ...      │   ...      │    ...     │
└─────────────────────────────────────────────────────────┘
```

---

## 二、样本（Sample）

### 2.1 定义

::: tip 核心定义
**样本（Sample）**：一行数据就是一个样本，所有样本的集合构成数据集。
:::

### 2.2 示例说明

```
数据集（6个样本）
    │
    ├── 样本1：[本科, 2年, 北京, 15000]
    ├── 样本2：[硕士, 3年, 上海, 20000]
    ├── 样本3：[本科, 1年, 深圳, 12000]
    ├── 样本4：[博士, 5年, 北京, 35000]
    ├── 样本5：[本科, 0年, 杭州, 8000]
    └── 样本6：[硕士, 2年, 广州, 18000]
```

::: info 记忆口诀
**一行一样本，多行成数据集**
:::

---

## 三、特征（Feature）

### 3.1 定义

::: tip 核心定义
**特征（Feature）**：一列数据称为一个特征，也称为属性（Attribute）。
:::

### 3.2 特征选择标准

| 分类 | 说明 | 示例 |
|------|------|------|
| ✅ **有效特征** | 对预测结果有帮助的信息 | 学历、工作经验、工作地点 |
| ❌ **无效特征** | 与预测结果无关的信息 | 学生编号、姓名 |

::: warning 注意
算法工程师统一使用"**特征**"而非"属性"，这是专业术语规范！
:::

### 3.3 特征示意

```python
# 特征矩阵 X
features = [
    [学历, 工作经验, 工作地点],  # 特征1, 特征2, 特征3
    ["本科", 2, "北京"],
    ["硕士", 3, "上海"],
    # ...
]
```

---

## 四、标签（Label）

### 4.1 定义

::: tip 核心定义
**标签（Label）**：要预测的目标列，也称为目标值（Target）。
:::

### 4.2 标签的动态性

标签根据**预测需求**动态确定：

| 预测任务 | 标签列 | 特征列 |
|----------|--------|--------|
| 预测薪资 | 就业薪资 | 学历、工作经验、工作地点 |
| 预测学科 | 培训学科 | 学历、工作经验、就业薪资 |

### 4.3 代码表示

```python
# 通用表示
X = [[特征1, 特征2, ...], ...]  # 特征矩阵
y = [标签1, 标签2, ...]          # 标签向量

# 实际示例
X = [["本科", 2, "北京"], ["硕士", 3, "上海"]]
y = [15000, 20000]  # 薪资标签
```

::: info 符号约定
- `x` 或 `X`：表示特征
- `y`：表示标签
:::

## 五、数据集划分

### 5.1 为什么要划分？

```
┌─────────────┐         ┌─────────────┐
│   训练集    │ ──────► │   模型训练   │
│ (Train Set) │         │  (学习规律)  │
└─────────────┘         └─────────────┘
                              │
                              ▼
┌─────────────┐         ┌─────────────┐
│   测试集    │ ──────► │   模型评估   │
│ (Test Set)  │         │  (验证效果)  │
└─────────────┘         └─────────────┘
```

### 5.2 常用划分比例

| 划分方案 | 训练集 | 测试集 | 适用场景 |
|----------|--------|--------|----------|
| **8:2** ⭐ | 80% | 20% | 最常用 |
| 7:3 | 70% | 30% | 次常用 |
| 6:2:2 | 60% 训练 | 20% 验证 | 20% 测试 |

### 5.3 符号表示

| 符号 | 含义 | 说明 |
|------|------|------|
| `X_train` | 训练集特征 | 用于模型学习的输入 |
| `y_train` | 训练集标签 | 用于模型学习的目标 |
| `X_test` | 测试集特征 | 用于验证的输入 |
| `y_test` | 测试集标签 | 用于验证的目标 |

### 5.4 代码示例

```python
from sklearn.model_selection import train_test_split

# 划分数据集（8:2）
X_train, X_test, y_train, y_test = train_test_split(
    X, y, 
    test_size=0.2,    # 测试集占 20%
    random_state=42   # 随机种子，保证可复现
)
```

## 六、概念对比总结

| 概念 | 英文 | 维度 | 说明 |
|------|------|------|------|
| 样本 | Sample | 行 | 一条完整的数据记录 |
| 特征 | Feature | 列 | 用于预测的输入变量 |
| 标签 | Label | 列 | 需要预测的目标变量 |
| 数据集 | Dataset | 表 | 所有样本的集合 |

---

## 七、知识小结

| 知识点 | 核心内容 | 考试重点/易混淆点 | 难度 |
|--------|----------|-------------------|------|
| 样本 | 一行数据即为一个样本 | 样本与数据集的关系（单行 vs 多行） | ⭐ |
| 特征 | 一列数据，需对预测结果有用 | 特征选择标准（与标签的关联性） | ⭐⭐ |
| 标签 | 需预测的目标值 | 标签与特征的动态对应关系 | ⭐ |
| 数据集划分 | 训练集 + 测试集 | 切分比例（8:2 或 7:3）及符号含义 | ⭐⭐ |
| 特征工程 | 特征需为对结果预测有用的信息 | 无效特征的识别（如编号列） | ⭐⭐⭐ |
