## 出题规律总结

### 高频主题
| 主题 | 出现年份 | 频次 |
|------|---------|------|
| 微服务/SOA | 2021、2024-11 | 2 |
| 数据集成/数据管理 | 2023、2024-11、2025-5 | 3 |
| 软件维护 | 2022、2024-11 | 2 |
| 软件测试 | 2020、2024-5、2025-5 | 3 |
| 云原生/云运维 | 2020、2024-5 | 2 |
| 分布式/事件驱动 | 2024-11、2025-5 | 2 |

### 趋势分析
1. **AI相关题目增加**：2025年5月出现AI测试用例生成
2. **分布式系统持续升温**：分布式事务、事件驱动架构
3. **数据管理深化**：从单一数据库到多模型数据管理
4. **运维自动化**：云上自动化运维、可观测性
