# 数据结构强化

## 第一章 

**时间复杂度和空间复杂度的计算**

**时间复杂度**：一个语句在算法中被重复执行的**次数**

常见形式：

1️⃣ 阶乘n!：1*2*...*n ，时间复杂度O(n)

2️⃣ 单个循环++：O(n)

3️⃣ 单个循环i*2: O(logn)

4️⃣ 双循环i与j无关联：O(n^2)

5️⃣ 双循环i与j关联：计算总次数

无论哪种形式 穷举各层的执行次数即可
