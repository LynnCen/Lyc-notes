# 计算机图形学原理 - 教学大纲与目录

## 系列概述

本系列是计算机图形学的理论基础教程，从数学基础到渲染算法，系统讲解图形学核心概念和经典算法。

## 目标读者

- 对图形学理论感兴趣的开发者
- 需要深入理解渲染原理的工程师
- 计算机图形学相关专业学生
- 游戏/可视化/CAD 开发者

## 前置知识

- 基础数学（高中水平）
- 编程基础
- 线性代数基础（有则更好）
