## 学习路径

```
计算机图形学原理（理论基础）
        ↓
    Canvas2D（入门）
        ↓
    WebGL（进阶）
        ↓
    PixiJS（实战）
        ↓
    无限画布（综合应用）
```

## 推荐顺序

1. **零基础**：先学习「计算机图形学原理」建立理论基础
2. **有基础**：直接从「Canvas2D 基础」开始实践
3. **想做游戏**：学习「PixiJS 实战」快速上手
4. **想做编辑器**：重点学习「无限画布」模块
