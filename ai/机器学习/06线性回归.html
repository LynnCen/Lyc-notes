1. 分析 ​

模型持久化概念：当训练数据量达到几百万甚至几千万时，每次预测都重新训练模型是不现实的，因此需要将训练好的模型参数保存下来，这个过程称为模型持久化。

实现方法：通过 joblib 库的 dump 和 load 方法实现，保存为二进制文件格式。

特征预处理 ​

归一化 ​

一、为什么做归一化和标准化 ​

  • 量纲问题：当特征的单位或大小相差较大时，数值较大的特征会主导模型训练结果。
  • 方差影响：某特征的方差相比其他特征大出几个数量级时，会导致模型无法有效学习其他特征。
  • 实例：男子健康数据中，身高(1.7m)、体重(67kg)、视力(1.5) 单位不同，体重数值明显大于其他特征，容易导致模型偏差。

例题：男子健康状况分析 ​

  • 特征：身高(米)、体重(kg)、视力(0.2–2.0)，单位差异明显。
  • 影响：体重范围(60–90) 远大于视力(0.6–1.8)，模型会误认为数值大的特征更重要。
  • 解决：通过归一化/标准化消除量纲影响，使各特征具有可比性。

二、归一化原理 ​

核心公式：将数据映射到 $[0, 1]$ 区间

$$ x_{norm} = \frac{x - x_{min}}{x_{max} - x_{min}} $$

扩展到任意区间 $[m_i, m_x]$：

$$ x_{scaled} = \frac{x - x_{min}}{x_{max} - x_{min}} \cdot (m_x - m_i) + m_i $$

计算步骤：

  1. 计算每列最小值和最大值
  2. 用上式得到 $[0, 1]$ 区间的值
  3. 若需其他区间，再用扩展公式做线性变换

示例：数值 90，$min=60$，$max=90$ 时，$(90-60)/(90-60)=1$；数值 60 则转换为 0。

标准化 ​

一、数据标准化 ​

核心概念：将原始数据转换为均值为 0、标准差为 1 的标准正态分布。

数学形式：

$$ x_{std} = \frac{x - \mu}{\sigma} $$

其中：

  • $\mu$（缪 /mu/）：均值，决定分布中心
  • $\sigma$（西格玛 /sigma/）：标准差，反映离散程度
  • $\sigma^2$：方差，标准差的平方

工程要点：

  • 面试时希腊字母发音要准确（μ: 缪，σ: 西格玛）
  • 标准化后约：68% 在 $[\mu-\sigma, \mu+\sigma]$，95% 在 $[\mu-2\sigma, \mu+2\sigma]$，99.7% 在 $[\mu-3\sigma, \mu+3\sigma]$

代码示例：

python
from sklearn.preprocessing import StandardScaler

transformer = StandardScaler()
data = transformer.fit_transform(data)

# 关键属性
transformer.mean_  # 各列均值
transformer.var_   # 各列方差（注意是方差，不是标准差）

线性回归简介与概念 ​

1. 核心要素与学习重点 ​

  • 核心要素：线性回归的分类与定义、API 使用、损失函数概念、评估方法。
  • 学习重点：损失函数在深度学习和面试中常被问到，需掌握模型训练使用的损失函数形式。

2. 例题：预测体重问题 ​

  • 思路：用已知身高–体重数据建立回归方程，预测新身高对应的体重。
  • 示例数据：5 组 (160cm/56.3kg, 166cm/60.6kg, 172cm/64.9kg, 178cm/69.2kg, 180cm/75kg)。
  • 预测目标：176cm 对应体重（示例预测约 65.4kg）。
  • 数学本质：寻找最佳拟合直线 $\hat{y} = kx + b$。
  • 特点：预测结果为近似值，非精确值。

3. 线性回归定义 ​

定义：用回归方程对**特征值（自变量）与目标值（因变量）**的关系建模。

数学表达：

  • 基础形式（一元）：$\hat{y} = wx + b$，其中 $w$ 为权重（斜率），$b$ 为偏置（截距）。
  • 多元形式：$\hat{y} = w_1 x_1 + w_2 x_2 + \cdots + w_d x_d + b$。
  • 矩阵形式：$\hat{\boldsymbol{y}} = \boldsymbol{X}\boldsymbol{w} + b$，涉及 $\boldsymbol{X}$ 的转置与维度约定（常把 $b$ 并入 $\boldsymbol{w}$，$\boldsymbol{X}$ 增加一列 1）。

应用示例：用房屋面积、位置、楼层、朝向等特征预测房价。

4. 线性回归分类 ​

类型说明示例
一元线性回归单特征仅用身高预测体重
多元线性回归多特征用面积、位置等多特征预测房价

实际工作中多元回归更常见。

5. 常见应用场景 ​

  • 钢轨伸缩长度与温度；昆虫鸣叫次数与天气；GDP 与双十一销售额等。
  • 共同点：特征与目标在散点图上呈大致线性趋势。

6. 概念小结 ​

  • 核心关系：特征值与目标值之间的线性关系，本质是 $\hat{y} = \boldsymbol{w}^{\top}\boldsymbol{x} + b$ 的变体。
  • 分类：一元 vs 多元。
  • 评估：回归用 MSE/MAE/RMSE 等，与分类（如 ACC）、聚类（如 CH）不同。

2. 损失函数 ​

基本概念：

  • 误差：$e_i = \hat{y}_i - y_i$（预测值减真实值）。
  • 损失函数：又称代价函数/目标函数，用于衡量预测效果；优化目标是找到使损失函数最小的参数。

最小二乘法（误差平方和）：

$$ J(w,b) = \sum_{i=1}^{m} (\hat{y}i - y_i)^2 = \sum^{m} (w x_i + b - y_i)^2 $$

固定 $b$ 后可展开为关于 $w$ 的二次函数，求导令 $\frac{\partial J}{\partial w}=0$ 得极值点；当二次项系数为正时，该点为最小值。

三种常用形式：

名称公式特点
MSE（均方误差）$\displaystyle \frac{1}{m}\sum_{i=1}^{m}(y_i - \hat{y}_i)^2$放大大误差，对异常值敏感
MAE（平均绝对误差）$\displaystyle \frac{1}{m}\sum_{i=1}^{m}\lvert y_i - \hat{y}_i\rvert$鲁棒，不受异常值主导
RMSE（均方根误差）$\displaystyle \sqrt{\frac{1}{m}\sum_{i=1}^{m}(y_i - \hat{y}_i)^2} = \sqrt{\text{MSE}}$量纲与 $y$ 一致

记忆：MSE = 误差平方和的均值；MAE = 误差绝对值的均值；RMSE = MSE 开根号。

3. 线性回归求解需要什么 ​

  • 数据：特征 $\boldsymbol{x}$ 与目标 $y$，满足大致线性关系；可为单特征或多特征。
  • 模型：假设函数 $\hat{y} = w x + b$（一元）或多元线性回归。
  • 损失：常用 MSE / MAE / RMSE；优化时多采用 MSE（最小二乘）。
  • 优化方法：梯度下降（迭代逼近，适合大规模）或正规方程（求偏导得解析解，适合小规模）。

正规方程 ​

1. 一元线性回归 ​

损失函数：衡量真实值与预测值差异，为优化指明方向。

$$ J(k, b) = \sum_{i=1}^{m} (\hat{y}i - y_i)^2 = \sum^{m} (k x_i + b - y_i)^2 $$

给定 5 组身高体重数据，将 $(x_i, y_i)$ 代入上式展开；对 $k$、$b$ 分别求偏导并令其为 0，联立解得最优 $k$、$b$。

2. 多元线性回归 ​

模型：$\hat{y} = \boldsymbol{w}^{\top}\boldsymbol{x} + b$，$\boldsymbol{w}$ 为权重向量，$\boldsymbol{x}$ 为 $d$ 维特征。
数据集：$\boldsymbol{X} \in \mathbb{R}^{m \times d}$，第 $i$ 行即第 $i$ 个样本；第 $i$ 个样本预测值 $\hat{y}_i = \boldsymbol{x}_i^{\top}\boldsymbol{w} + b$（常将 $b$ 并入 $\boldsymbol{w}$，$\boldsymbol{X}$ 增加一列 1）。
损失函数（最小二乘）：

$$ J(\boldsymbol{w}) = |\boldsymbol{X}\boldsymbol{w} - \boldsymbol{y}|^2 = (\boldsymbol{X}\boldsymbol{w} - \boldsymbol{y})^{\top}(\boldsymbol{X}\boldsymbol{w} - \boldsymbol{y}) $$

正规方程解：对 $\boldsymbol{w}$ 求导令其为 0，得（在 $\boldsymbol{X}^{\top}\boldsymbol{X}$ 可逆时）：

$$ \boldsymbol{w}^* = (\boldsymbol{X}^{\top}\boldsymbol{X})^{-1} \boldsymbol{X}^{\top} \boldsymbol{y} $$

房价预测应用要点：特征矩阵 $\boldsymbol{X}$ 常含面积、卧室数、房龄等，第一列可设为 1（对应截距）；先算 $\boldsymbol{X}^{\top}\boldsymbol{X}$，求逆后与 $\boldsymbol{X}^{\top}\boldsymbol{y}$ 相乘得 $\boldsymbol{w}$。

3. 总结 ​

  • 核心公式：$\boldsymbol{w} = (\boldsymbol{X}^{\top}\boldsymbol{X})^{-1}\boldsymbol{X}^{\top}\boldsymbol{y}$，一步得到最优解。
  • 特点：无需迭代、不需学习率；适合特征维度不高；高维时求逆代价大。
  • 与梯度下降：正规方程直接解析解；梯度下降多步迭代逼近。

4. 知识小结（正规方程） ​

知识点核心内容考试重点/易混淆点难度
正规方程 vs 梯度下降正规方程一步求解析解；梯度下降迭代逼近效率与适用规模⭐⭐⭐
最小二乘与偏导损失为误差平方和，对 k、b 求偏导联立推导过程⭐⭐
正规方程推导$\boldsymbol{w}=(\boldsymbol{X}^{\top}\boldsymbol{X})^{-1}\boldsymbol{X}^{\top}\boldsymbol{y}$$\boldsymbol{X}^{\top}\boldsymbol{X}$ 需可逆⭐⭐⭐⭐
多元与矩阵权重向量、特征矩阵、转置与维度维度匹配⭐⭐⭐

梯度下降 ​

1. 梯度与梯度下降 ​

梯度定义：函数在某点变化率最快的方向。

  • 单变量：梯度即导数（切线斜率），表示函数增长最快的方向。
  • 多变量：梯度为各变量偏导数组成的向量 $\nabla J = (\frac{\partial J}{\partial \theta_1}, \ldots, \frac{\partial J}{\partial \theta_d})^{\top}$。

参数更新公式：

$$ \theta_{new} = \theta_{old} - \alpha \cdot \frac{\partial J}{\partial \theta} $$

  • $\alpha$：学习率（步长），常取 0.001～0.01；过大易震荡，过小收敛慢。
  • 负号：梯度是上升方向，要最小化故沿负梯度方向更新。

2. 单变量梯度下降示例 ​

例题：求 $J(\theta) = \theta^2$ 的极小值。

  • 导数：$\frac{dJ}{d\theta} = 2\theta$。
  • 设初始 $\theta_0 = 1$，学习率 $\alpha = 0.4$。
  • 更新：$\theta_{t+1} = \theta_t - 0.4 \times 2\theta_t = 0.2,\theta_t$。
  • 迭代：1 → 0.2 → 0.04 → 0.008 → …，趋近 0。
  • 收敛判断：$|\theta_{new} - \theta_{old}|$ 或 $|\nabla J|$ 足够小则停止。

3. 多变量梯度下降示例 ​

例题：$J(\theta_1, \theta_2) = \theta_1^2 + \theta_2^2$。

  • 偏导：$\frac{\partial J}{\partial \theta_1} = 2\theta_1$，$\frac{\partial J}{\partial \theta_2} = 2\theta_2$。
  • 梯度向量：$\nabla J = (2\theta_1,, 2\theta_2)^{\top}$。
  • 取起点 $(1, 3)$，$\alpha = 0.1$：
    • 第一步：$(1,3) - 0.1(2,6) = (0.8, 2.4)$；
    • 第二步：$(0.8, 2.4) - 0.1(1.6, 4.8) = (0.64, 1.92)$；
    • 持续迭代趋近 $(0, 0)$。

4. 知识小结（梯度下降基础） ​

知识点核心内容考试重点/易混淆点难度
梯度概念单变量=导数；多变量=偏导向量；沿梯度方向函数上升最快负梯度方向用于最小化⭐⭐
单变量示例$J=\theta^2$，$dJ/d\theta=2\theta$，$\theta \leftarrow \theta - \alpha \cdot 2\theta$初始值、学习率⭐⭐
多变量示例$J=\theta_1^2+\theta_2^2$，偏导后同步更新 $\theta_1,\theta_2$同时更新所有参数⭐⭐⭐
通用原理迭代式 $\theta \leftarrow \theta - \alpha \nabla J$，学习率控制步长学习率过大/过小⭐⭐
符号损失可用 $J(\theta)$ 或 $L(\theta)$，无本质区别不混淆即可⭐

5. 梯度下降基本步骤与负号、学习率 ​

四步流程：① 初始化参数与学习率 $\alpha$；② 计算当前损失对参数的梯度 $\nabla J$；③ 更新 $\theta \leftarrow \theta - \alpha \nabla J$；④ 若 $|\theta_{new}-\theta_{old}|<\varepsilon$ 或达到最大迭代次数则停止。

负号意义：梯度方向是函数增加最快的方向；要最小化损失，故沿负梯度方向更新（“下山”选最陡下坡）。考题常问：“为什么用负梯度？”——因为我们要的是下降方向。

学习率：

  • 过小：收敛慢、易陷局部最优。
  • 过大：可能越过最优点（Overshooting）、震荡（Oscillation），甚至发散。
  • 常用范围约 0.001～0.01。

6. 应用案例：银行信贷（梯度计算） ​

设定：3 个特征（月薪、存款、房产面积），目标为授信额度；8 条样本。
假设函数：$\hat{y} = w_0 + w_1 x_1 + w_2 x_2 + w_3 x_3$（$x_0=1$ 表截距）。
损失函数（带 1/2 便于求导）：

$$ J(\boldsymbol{w}) = \frac{1}{2m}\sum_{i=1}^{m}(\hat{y}_i - y_i)^2 $$

求导后平方项产生因子 2，与分母 2 抵消，梯度形式更简洁。
向量形式：梯度 $\nabla J = \frac{1}{m}\boldsymbol{X}^{\top}(\boldsymbol{X}\boldsymbol{w} - \boldsymbol{y})$；标量形式对第 $j$ 个权重：$\frac{\partial J}{\partial w_j} = \frac{1}{m}\sum_{i=1}^{m}(\hat{y}i - y_i)x$。
更新：$w_j \leftarrow w_j - \alpha \cdot \frac{\partial J}{\partial w_j}$。批量法每轮用全部样本算平均梯度再更新；迭代多轮直至收敛。

7. 梯度下降法分类 ​

类型梯度计算方式公式（示意）特点
全梯度下降 FGD使用全部样本梯度求平均$\theta \leftarrow \theta - \alpha \frac{1}{m}\sum_{i=1}^{m}\nabla J_i$稳定、计算量大
随机梯度下降 SGD每轮随机选 1 个样本的梯度$\theta \leftarrow \theta - \alpha \nabla J_i$快、波动大
小批量 Mini-batch每轮用一小批样本（如 32）$\theta \leftarrow \theta - \alpha \frac{1}{\lvert B\rvert}\sum_{i \in B}\nabla J_i$折中，常用
随机平均梯度 SAG用当前样本梯度与历史梯度均值维护梯度表，用均值更新后期收敛好，需存历史
  • FGD：Full，全样本；适合小数据。
  • SGD：Stochastic，单样本；应用广但不稳定。
  • Mini-batch：batch_size=1 即 SGD，=m 即 FGD。
  • SAG：唯一用到“历史梯度”的；初期可能慢（初始梯度为 0）。

知识点核心内容考试重点/易混淆点难度
梯度下降原理沿负梯度迭代 $\theta \leftarrow \theta - \alpha \nabla J$为什么取负梯度⭐⭐⭐⭐
损失函数 1/2$J=\frac{1}{2m}\sum(\hat{y}-y)^2$，求导后系数简洁加 1/2 的目的⭐⭐⭐

9. 梯度下降与正规方程对比 ​

对比项正规方程梯度下降
公式$\boldsymbol{w}=(\boldsymbol{X}^{\top}\boldsymbol{X})^{-1}\boldsymbol{X}^{\top}\boldsymbol{y}$$\theta \leftarrow \theta - \alpha \nabla J$
学习率不需要需要 $\alpha$
计算方式一步求解析解迭代逼近
适用小数据、特征数不多大数据、高维特征（如 28×28 图像）
缺点需 $\boldsymbol{X}^{\top}\boldsymbol{X}$ 可逆、求逆贵、对噪声敏感学习率需调、可能震荡

小结：梯度 = 单变量导数/多变量偏导向量；各类 GD（FGD/SGD/Mini-batch/SAG）只是“用哪批样本算梯度”不同，更新形式统一为 $\theta \leftarrow \theta - \alpha \cdot$（梯度）。

回归模型评估方法 ​

目的：用指标衡量预测与真实的差距，判断拟合效果。公式与损失函数相同，但用途是评估/选模型，而非训练时的优化目标。

1. 三种指标公式 ​

指标全称公式特性
MSEMean Squared Error$\displaystyle \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$值越小越准；对异常值敏感（误差>1 被放大）
MAEMean Absolute Error$\displaystyle \frac{1}{n}\sum_{i=1}^{n}\lvert y_i - \hat{y}_i\rvert$稳定，对异常值不敏感
RMSERoot Mean Squared Error$\displaystyle \sqrt{\frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2} = \sqrt{\text{MSE}}$量纲与 $y$ 一致；同样敏感于异常值

3. 应用案例要点 ​

  • 噪声拟合：在真实关系（如 $y=2x+5$）上加正态噪声 $\mathcal{N}(0,1)$，通常 RMSE > MAE（平方放大效应）。

方法公式API特点

欠拟合与过拟合 ​

1. 概念与误差表现 ​

类型定义误差表现
欠拟合模型过于简单，训练/测试都差训练误差大、测试误差大
过拟合模型过于复杂，记住噪声训练误差小、测试误差大

图形记忆：横轴模型复杂度，纵轴错误率；左侧欠拟合（双高），中间最优点，右侧过拟合（训练低、测试高）。

2. 原因与解决方案 ​

欠拟合原因：模型/特征过于简单、学习能力不足。
解决：增加特征（如 $x^2,, x^3$）、用更复杂模型、特征组合；注意过度会变成过拟合。

过拟合原因：特征冗余、数据噪声（bad case）、模型过复杂、数据量不足。
解决：① 清洗数据（分析 bad case）；② 增加数据量；③ 正则化（L1/L2）；④ 降维（如 PCA）。
工程顺序：清洗 → 增数据 → 正则化 → 降维。

3. 代码示例：欠拟合 / 正常拟合 / 过拟合 ​

数据：$x \sim \text{Uniform}(-3,3)$，$y = 2x^2 + 3x + 1 + \varepsilon$，$\varepsilon \sim \mathcal{N}(0,1)$。

python
import numpy as np
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error
from sklearn.preprocessing import PolynomialFeatures

np.random.seed(42)
X = np.random.uniform(-3, 3, size=100).reshape(-1, 1)
y = 2 * X.ravel()**2 + 3 * X.ravel() + 1 + np.random.normal(0, 1, 100)

# 欠拟合：仅线性
lr = LinearRegression().fit(X, y)
print('欠拟合 MSE:', mean_squared_error(y, lr.predict(X)))  # 约 2.4

# 正常拟合：加二次项，模型 y = w0 + w1*x + w2*x^2
X2 = np.hstack([X, X**2])
lr2 = LinearRegression().fit(X2, y)
print('二次拟合 MSE:', mean_squared_error(y, lr2.predict(X2)))  # 约 0.96

# 过拟合：高次项 X, X^2, ..., X^9
X9 = np.column_stack([X**i for i in range(1, 10)])
lr9 = LinearRegression().fit(X9, y)
# 训练 MSE 更小，但测试集或新数据上会变差

4. 正则化（L1 / L2） ​

目的：在损失函数中加惩罚项，约束权重大小，减轻异常点影响，相当于对特征做筛选或平滑。

L1 正则化（Lasso）：

$$ J(\boldsymbol{w}) = \frac{1}{2m}\sum_{i=1}^{m}(\hat{y}i - y_i)^2 + \alpha \sum^{d}|w_j| $$

  • 对 $|w_j|$ 求导：$w_j>0$ 时导数为 $+1$，$w_j<0$ 时为 $-1$，$w_j=0$ 处不可导（次梯度）。
  • 效果：权重被压向 0，部分权重精确为 0，实现特征筛选（稀疏解）。

L2 正则化（Ridge）：

$$ J(\boldsymbol{w}) = \frac{1}{2m}\sum_{i=1}^{m}(\hat{y}i - y_i)^2 + \alpha \sum^{d} w_j^2 $$

  • 对 $w_j$ 求导多出一项 $2\alpha w_j$，权重整体缩小但一般不为 0。
  • 效果：权值平滑、减小过拟合，不产生稀疏。

对比：L1 稀疏、可做特征选择；L2 平滑、数值稳定。工程上过拟合排查：先 bad case → 数据量 → 正则化 → 降维。

正常拟合适当复杂度（如二次项），训练与测试误差均衡MSE 从 2.4 降到 0.96⭐⭐
