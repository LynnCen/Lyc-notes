全连接神经网络（FNN / FCN） ​

全连接神经网络（Fully Connected Network，FCN），又称前馈神经网络（Feedforward Neural Network，FNN），是深度学习最基础的网络结构。每层的每个神经元都与上一层所有神经元相连，因此得名"全连接"。

文档结构： 网络结构 → 激活函数 → 损失函数 → 前向传播 → 反向传播（链式法则）→ 梯度消失与爆炸 → 梯度下降 → 优化器 → 正则化 → 知识小结。

一、网络结构 ​

三层组成 ​

x1 ──┐          [h1]─┐              y1
x2 ──┼── W1 ──→ [h2] ├── W2 ──→    y2
x3 ──┘          [h3]─┘              y3

层作用
输入层接收原始特征 $x_1, x_2, \ldots, x_n$
隐藏层进行特征变换，层数越多、神经元越多，表达能力越强
输出层输出预测结果（回归输出连续值，分类输出概率）

单个神经元的计算 ​

$$z = \boldsymbol{w}^{\top}\boldsymbol{x} + b = \sum_{i=1}^{n} w_i x_i + b$$

$$a = f(z)$$

其中 $f$ 是激活函数。增加隐藏层数、增加神经元数均可增强网络的表达能力。

二、激活函数 ​

激活函数引入非线性，没有激活函数的多层网络等价于单层线性变换，无法拟合复杂函数。

1. 阶跃函数 / 符号函数（Sign） ​

$$f(z) = \begin{cases} +1 & z \geq 0 \ -1 & z < 0 \end{cases}$$

值域：${-1, +1}$；不可导，只用于感知机，深度学习中不使用。

2. Sigmoid ​

$$\sigma(z) = \frac{1}{1 + e^{-z}}$$

值域：$(0, 1)$；导数：$\sigma'(z) = \sigma(z)(1-\sigma(z))$，最大值为 0.25。

优点：输出可解释为概率，适合二分类输出层。

缺点：深层网络易发生梯度消失（导数最大仅 0.25，多层连乘趋近 0）；含指数，计算略慢。

3. Tanh（双曲正切） ​

$$\tanh(z) = \frac{e^z - e^{-z}}{e^z + e^{-z}}$$

值域：$(-1, 1)$；导数：$\tanh'(z) = 1 - \tanh^2(z)$，最大值为 1。

优点：零中心化（均值为 0），比 Sigmoid 收敛快。

缺点：仍存在梯度消失问题。

4. ReLU（Rectified Linear Unit） ​

$$f(z) = \max(0, z) = \begin{cases} z & z > 0 \ 0 & z \leq 0 \end{cases}$$

值域：$[0, +\infty)$；导数：$z>0$ 时为 1，$z\leq0$ 时为 0。

优点：计算简单；正区间梯度为 1，缓解梯度消失；收敛快。

缺点："死亡 ReLU" 问题——负区间梯度恒为 0，神经元可能永久失活。

5. Swish ​

$$f(z) = z \cdot \sigma(z) = \frac{z}{1 + e^{-z}}$$

优点：平滑、非单调；在深层网络（如 EfficientNet）中表现优于 ReLU。

6. Softmax（多分类输出层） ​

$$\text{Softmax}(z_k) = \frac{e^{z_k}}{\sum_{j=1}^{K} e^{z_j}}$$

值域：每个输出在 $(0,1)$ 且所有输出之和为 1，可解释为各类别概率。只用于多分类输出层。

激活函数对比 ​

函数值域梯度消失适用场景
Sign${-1,+1}$不适用感知机
Sigmoid$(0,1)$严重二分类输出层
Tanh$(-1,1)$较重隐藏层（浅层）
ReLU$[0,+\infty)$轻微隐藏层（首选）
Swish$\approx(-0.28,+\infty)$极轻深层网络
Softmax$(0,1)$，和为 1不适用多分类输出层

三、损失函数 ​

衡量预测值与真实值之间的差距，是训练的优化目标。

1. 均方误差（MSE）—— 回归任务 ​

$$\mathcal{L} = \frac{1}{n}\sum_{i=1}^{n}(\hat{y}_i - y_i)^2$$

对大误差惩罚强（平方放大）；对异常值敏感。

2. 交叉熵损失（Cross-Entropy）—— 分类任务 ​

二分类：

$$\mathcal{L} = -\frac{1}{n}\sum_{i=1}^{n}\big[y_i\log\hat{y}_i + (1-y_i)\log(1-\hat{y}_i)\big]$$

多分类（配合 Softmax）：

$$\mathcal{L} = -\frac{1}{n}\sum_{i=1}^{n}\sum_{k=1}^{K} y_{ik}\log\hat{y}_{ik}$$

凸函数，由最大似然推导而来，梯度计算简洁。

选择建议 ​

任务类型输出激活损失函数
回归线性（无激活）MSE
二分类Sigmoid二分类交叉熵
多分类Softmax多分类交叉熵

四、前向传播（Forward Pass） ​

前向传播是将输入数据沿网络依次计算，得到最终预测值的过程。

以两层网络（输入→隐藏→输出）为例：

$$\boldsymbol{z}^{(1)} = \boldsymbol{W}^{(1)}\boldsymbol{x} + \boldsymbol{b}^{(1)}$$

$$\boldsymbol{a}^{(1)} = f(\boldsymbol{z}^{(1)})$$

$$\boldsymbol{z}^{(2)} = \boldsymbol{W}^{(2)}\boldsymbol{a}^{(1)} + \boldsymbol{b}^{(2)}$$

$$\hat{\boldsymbol{y}} = g(\boldsymbol{z}^{(2)})$$

其中 $f$ 是隐藏层激活函数，$g$ 是输出层激活函数（如 Softmax）。前向传播结束后计算损失 $\mathcal{L}(\hat{\boldsymbol{y}}, \boldsymbol{y})$。

五、反向传播（Backpropagation） ​

反向传播利用链式法则，从损失函数出发，逐层计算每个参数对损失的梯度，再用梯度下降更新参数。

链式法则 ​

复合函数 $z = f(g(x))$ 的导数：

$$\frac{dz}{dx} = \frac{dz}{dy} \cdot \frac{dy}{dx}, \quad y = g(x)$$

反向传播流程 ​

text
① 前向传播，保存每层的 z 和 a
② 计算输出层对损失的梯度 dL/dz_out
③ 逐层向后（链式法则）：
   dL/dW = dL/dz · a_prev（权重梯度）
   dL/db = dL/dz（偏置梯度）
   dL/da_prev = dL/dz · W（传给上一层）
④ 更新参数：W := W - α * dL/dW

六、梯度消失与梯度爆炸 ​

梯度消失（Vanishing Gradient） ​

原因：链式求导时，若每层激活函数的梯度 $\lvert\partial a/\partial z\rvert < 1$（如 Sigmoid 最大 0.25），多层连乘后梯度指数级缩小，前层参数几乎不更新。

解决方案：

方法原理
使用 ReLU / Swish正区间梯度为 1，不衰减
Batch Normalization归一化每层激活，稳定梯度
残差连接（ResNet）梯度可直接跳过若干层流动
LSTM / GRU门控机制保留远距离梯度信息

梯度爆炸（Exploding Gradient） ​

原因：权重较大时，链式乘积指数级增大，参数更新量过大导致不收敛。

解决方案：

方法原理
梯度裁剪（Gradient Clipping）超过阈值时按比例缩放：$g \leftarrow g \cdot \frac{c}{\lVert g\rVert}$
权重正则化（L1/L2）惩罚过大的权重值
He / Xavier 初始化控制初始梯度规模

七、梯度下降算法 ​

目标：找到最优参数 $\boldsymbol{w}^*$，使损失函数 $\mathcal{L}$ 最小。

$$\boldsymbol{w} \leftarrow \boldsymbol{w} - \alpha \nabla_{\boldsymbol{w}} \mathcal{L}$$

三种变体 ​

类型每次使用样本数优点缺点
批量梯度下降 BGD全部 $n$ 个稳定，可收敛到全局每步计算量大，慢
随机梯度下降 SGD1 个快，可跳出局部最优噪声大，不稳定
小批量 Mini-batch GD$b$ 个（如 32/64/128）折中，GPU 并行友好需调 batch size

实际训练中几乎都用 Mini-batch GD。

八、优化器 ​

优化器在梯度下降基础上改进，加快收敛、提高稳定性。

1. 标准 SGD ​

$$\boldsymbol{w} \leftarrow \boldsymbol{w} - \alpha \boldsymbol{g}$$

缺点：学习率固定，平坦区域收敛慢，峡谷区域震荡。

2. 动量优化器（Momentum） ​

积累历史梯度方向：

$$\boldsymbol{v} \leftarrow \beta \boldsymbol{v} - \alpha \boldsymbol{g}, \quad \boldsymbol{w} \leftarrow \boldsymbol{w} + \boldsymbol{v}$$

  • $\beta$（动量系数，常取 0.9）：历史方向保留比例。
  • 优点：梯度稳定方向越来越快；有助于跳出局部极值。
  • 缺点：需手动设置学习率和动量系数。

3. Adagrad ​

对不同参数自适应学习率，历史梯度大的参数学习率小：

$$G \leftarrow G + g^2, \quad w \leftarrow w - \frac{\alpha}{\sqrt{G + \varepsilon}} g$$

  • 优点：自适应学习率；对稀疏特征（如 NLP 词嵌入）友好。
  • 缺点：$G$ 单调递增，学习率最终趋近 0，训练后期失效。

4. RMSprop ​

对 Adagrad 改进，引入指数加权移动平均：

$$G \leftarrow \rho G + (1-\rho) g^2, \quad w \leftarrow w - \frac{\alpha}{\sqrt{G + \varepsilon}} g$$

  • $\rho$ 常取 0.9；对 RNN 效果好。
  • 优点：解决 Adagrad 学习率消失；适合非平稳目标。

5. Adam（自适应矩估计） ​

同时维护一阶矩（梯度均值）和二阶矩（梯度方差）：

$$m \leftarrow \beta_1 m + (1-\beta_1) g$$

$$v \leftarrow \beta_2 v + (1-\beta_2) g^2$$

偏差修正（补偿初始化为 0 的偏差）：

$$\hat{m} = \frac{m}{1-\beta_1^t}, \quad \hat{v} = \frac{v}{1-\beta_2^t}$$

参数更新：

$$w \leftarrow w - \frac{\alpha}{\sqrt{\hat{v}} + \varepsilon} \hat{m}$$

  • 默认：$\beta_1=0.9$，$\beta_2=0.999$，$\varepsilon=10^{-8}$，$\alpha=10^{-3}$。
  • 优点：收敛快，对超参数不敏感，适合大多数场景。

优化器对比 ​

优化器自适应学习率动量推荐场景
SGD否否简单任务
Momentum否是需要加速的场景
Adagrad是否稀疏特征（词嵌入）
RMSprop是否RNN / 非平稳目标
Adam是是大多数深度学习任务（默认首选）

九、正则化 ​

防止过拟合的关键手段，通过约束模型复杂度提升泛化能力。

过拟合 vs 欠拟合 ​

问题表现原因解决
欠拟合训练/测试误差都大模型太简单增加层数、神经元；减小正则化强度
过拟合训练误差小、测试误差大模型太复杂或数据太少正则化、Dropout、数据增强、早停

常见正则化方法 ​

1. L2 正则化（权重衰减，Ridge）：

$$\mathcal{L}_{total} = \mathcal{L} + \frac{\lambda}{2n}\sum_j w_j^2$$

权重更新多了衰减项：$w \leftarrow w(1 - \alpha\lambda/n) - \alpha\partial\mathcal{L}/\partial w$。使权重趋于小值但不为 0。

2. L1 正则化（Lasso）：

$$\mathcal{L}_{total} = \mathcal{L} + \frac{\lambda}{n}\sum_j \lvert w_j\rvert$$

产生稀疏解（部分权重恰好为 0），等效于特征选择。

3. Dropout：

训练时随机将神经元输出置为 0（丢弃概率 $p$），预测时乘以 $(1-p)$ 缩放。等效于集成大量子网络，隐藏层常用 $p=0.5$。

4. Batch Normalization（BN）：

对每个 mini-batch 的激活值归一化（均值为 0、方差为 1），再通过可学习的 $\gamma, \beta$ 缩放：

$$\hat{z} = \frac{z - \mu_B}{\sqrt{\sigma_B^2 + \varepsilon}}, \quad a = \gamma\hat{z} + \beta$$

效果：稳定训练；支持更大学习率；有正则化效果（可少用 Dropout）。

5. 早停（Early Stopping）：

监控验证集损失，不再下降时提前结束训练，避免继续过拟合。

十、知识小结 ​

知识点核心内容考试重点/易混淆点难度
网络结构输入层→隐藏层→输出层；全连接即每层所有神经元相互相连与卷积等稀疏连接区别⭐
激活函数引入非线性；ReLU 是隐藏层首选；Softmax 用于多分类输出Sigmoid/Tanh 易梯度消失⭐⭐
损失函数回归用 MSE，分类用交叉熵损失函数（优化）vs 评估指标⭐⭐
前向传播从输入逐层算到输出；保存中间值供反向用矩阵乘法 + 激活顺序⭐⭐
反向传播链式法则逐层求梯度；从输出层向输入层反推为什么保存前向中间值⭐⭐⭐
梯度消失多层 Sigmoid 乘积趋零；解决：ReLU、BN、残差连接权重小于 1 时连乘趋零⭐⭐⭐
梯度爆炸权重大时连乘梯度过大；解决：梯度裁剪、正则化梯度裁剪公式⭐⭐⭐
